# Project Organizer Tool

This is a project organizer tool used in the "Node.js.

## Technologies
* Node.js
* Express
* Express Messages, Session, Connect Flash & Validation
* MongoDB & Mongoose
* Pug Templating
* Passport.js Authentication
* BCrypt Hashing


### Installation

Install the dependencies

```sh
$ npm install
```
Run app

```sh
$ npm start
```
